## 🎮 The Story

This game was built entirely by AI in a single conversation. One human said *"let's make a Windows simulator where everything errors and help is useless"* — and an AI agent wrote 1,300 lines of pure chaos in one shot, deployed it live, and you're looking at the result.

No frameworks. No dependencies. No build step. Just one HTML file and a dream (nightmare).

## 💀 What Is This?

A fully interactive fake Windows 95 desktop where:

- **Every click** has a ~30% chance of spawning an error dialog
- **Error buttons** (OK, Cancel, Retry) cause *more* errors
- **The Help page** exists solely to mock you
- **Clippy** appears constantly with deeply unhelpful advice
- **The Calculator** gives wrong answers on purpose
- **Notepad** corrupts your text with random unicode
- **Paint** glitches out while you draw
- **Internet Explorer** takes forever and then everything is broken
- **Desktop icons** rearrange themselves randomly
- **System stability** drops until you hit a **Blue Screen of Death**
- **3 BSODs = Game Over.** How long can you survive?

## 🎯 How to Play

Open it. Click things. Try not to crash. You will crash.

## 🛠️ What's Inside

- Full windowing system with drag, minimize, maximize, close
- 8+ interactive applications (all cursed)
- Escalating chaos engine — errors breed more errors
- Sound effects (startup chime, error beeps, BSOD buzz)
- Screen shake on errors
- Clippy AI assistant (he doesn't help)
- Start menu, taskbar with clock, desktop icons
- Blue Screen of Death with fake memory dumps
- Game over screen with survival stats and rating (you'll get F---)
- Mobile + desktop support
